# SlotSpot
## Backend

- **Node.js** + **TypeScript**
- **Fastify** — HTTP-фреймворк
- **Drizzle ORM** — работа с базой данных
- **PostgreSQL 15** — база данных
- **Docker** — контейнеризация

# PostgreSQL 15 разворачивается в Docker-контейнере:

```bash
cd ./pkg/ecosystem
docker compose up -d

```
